# 文档入口（单一真源）

本目录内的文档是仓库的唯一真源。任何 AI/协作者必须先读本页。

## 活文档（唯一真源）

- `PROJECT_STATUS.md`：当前进度、已完成/进行中/下一步、风险与验证方式
- `ARCHITECTURE.md`：架构与数据流（离线/同步/权限/关键存储）
- `MODULES.md`：模块清单与代码入口定位（以代码为准）
- `WORKFLOW.md`：协作工作流（提交节奏、更新规则、避免回滚）
- `AI_RULES.md`：跨 IDE 的 AI 行为准则（.windsurf/.kiro 等都应遵循）

## 约束

- 文档不得编造代码中不存在的功能。
- 任何代码改动都必须同步更新 `PROJECT_STATUS.md`。
